Comandos git

Git com Repositório Local

// Configuração inicial do Git git config --global user.name "Seu Nome" git config --global user.email "seuemail@email.com" git config --list // Iniciar o versionamento na pasta atual git init // Verificar o estado dos arquivos git status // Adicionar todos os arquivos para a área de stage git add . ou git add [nome-do-arquivo] // Registrar as alterações no histórico git commit -m "mensagem descritiva" // Ver histórico de commits git log // Ver histórico resumido (por branch) git log --oneline --graph --decorate // Criar uma nova branch git branch [nome-da-branch] // Listar todas as branchs git branch // Mudar para outra branch git switch [nome-da-branch] // Unir uma branch à principal (geralmente “main” ou “master”) git merge [nome-da-branch]

Git com GitHub

// Clonar um repositório remoto git clone [url-do-repositório] // Entrar na pasta clonada cd [nome-da-pasta] // Verificar o repositório remoto conectado git remote -v // Criar uma nova branch git branch [nome-da-branch] // Mudar para a nova branch git switch [nome-da-branch] // Adicionar arquivos alterados git add . // Registrar as alterações git commit -m "mensagem descritiva" // Enviar alterações da branch atual para o GitHub git push origin [nome-da-branch] // Atualizar seu repositório local com mudanças do remoto git pull origin [nome-da-branch]
